## 📊 資料流程

```
用戶輸入
  ├─ 契約容量設定
  ├─ 電價方案選擇
  └─ 負載資料生成/上傳
       │
       ↓
時段判定 (get_tou_series)
  ├─ 尖峰/半尖峰/離峰標記
  └─ 夏月/非夏月標記
       │
       ↓
EMS 模擬 (run_ems_simulation)
  ├─ 充放電策略
  ├─ SOC 追蹤
  └─ BESS 功率計算
       │
       ↓
財務計算 (calculate_15y_logic)
  ├─ 電價套利收益
  ├─ DR 收益
  ├─ SOH 衰減
  └─ 15 年現金流
       │
       ↓
結果視覺化
  ├─ 負載曲線圖
  ├─ 現金流圖表
  └─ ROI 矩陣
```
